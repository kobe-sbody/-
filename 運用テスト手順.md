# 実運用テスト手順

3本のカウンセリング動画で、評価レポートの精度とトーンを確認するための手順です。

---

## 事前準備（1回だけ）

```bash
cd /Users/yuma/work/scratch/pilates-session-review
source .venv/bin/activate
```

`.env` に `OPENAI_API_KEY` が設定済みであることを確認してください。

---

## 1. 動画ファイルを置く場所

```
pilates-session-review/input/スタッフ名/日付_初回.mp4
```

**例（3人分）:**

```
input/
├── 田中/2025-05-26_初回.mp4
├── 佐藤/2025-05-28_初回.mp4
└── 山田/2025-05-30_初回.mp4
```

1人が3本テストする場合:

```
input/田中/2025-05-26_初回.mp4
input/田中/2025-05-28_2回目.mp4
input/田中/2025-06-01_3回目.mp4
```

---

## 2. 実行コマンド

ターミナルで、**1本ずつ** 以下を実行します。

```bash
cd /Users/yuma/work/scratch/pilates-session-review
source .venv/bin/activate

python run_counseling.py \
  input/田中/2025-05-26_初回.mp4 \
  --staff 田中 \
  --date 2025-05-26 \
  --output reports/田中/2025-05-26-report.html
```

**ポイント:**
- `input/...` → 置いた動画のパス
- `--staff` → スタッフ名（レポートに表示）
- `--date` → セッション日
- `--output` → レポートの保存先（スタッフ別フォルダ推奨）

2本目・3本目も同様に、`--output` のファイル名だけ変えて実行。

---

## 3. レポートの出力先

```
reports/スタッフ名/日付-report.html
```

**例:**
```
reports/田中/2025-05-26-report.html
reports/佐藤/2025-05-28-report.html
```

Cursor 左サイドバー → `reports` フォルダから開くか:

```bash
open reports/田中/2025-05-26-report.html
```

---

## 4. LINE用テキストの出力先

レポートと同じ名前で、拡張子が `.line.txt` のファイルです。

```
reports/スタッフ名/日付-report.line.txt
```

**例:**
```
reports/田中/2025-05-26-report.line.txt
```

**使い方:**
1. ファイルを開いて全文コピー → LINEに貼り付け
2. または HTML レポートの「LINE用テキストをコピー」ボタン

文字起こし全文:
```
reports/スタッフ名/日付-report-transcript.txt
```
（`run_counseling.py` 実行時は `counseling-report-transcript.txt` 形式）

---

## 5. 複数スタッフ・複数動画の管理

### フォルダで分ける（推奨・追加機能不要）

| 種類 | 置き場所 |
|------|---------|
| 入力動画 | `input/スタッフ名/日付_初回.mp4` |
| HTMLレポート | `reports/スタッフ名/日付-report.html` |
| LINE用 | `reports/スタッフ名/日付-report.line.txt` |
| 文字起こし | `reports/スタッフ名/日付-report-transcript.txt` |

### 実行前にレポート用フォルダを作る

```bash
mkdir -p reports/田中 reports/佐藤 reports/山田
```

### 3本テスト用コマンド例

```bash
# 1本目
python run_counseling.py input/田中/2025-05-26_初回.mp4 \
  --staff 田中 --date 2025-05-26 \
  --output reports/田中/2025-05-26-report.html

# 2本目
python run_counseling.py input/田中/2025-05-28_2回目.mp4 \
  --staff 田中 --date 2025-05-28 \
  --output reports/田中/2025-05-28-report.html

# 3本目
python run_counseling.py input/田中/2025-05-30_3回目.mp4 \
  --staff 田中 --date 2025-05-30 \
  --output reports/田中/2025-05-30-report.html
```

### 確認チェックリスト（現場テスト用）

各レポートについてメモ:

- [ ] スコアは体感と合っているか
- [ ] 良かった点は具体的か
- [ ] 改善ポイントは的確か
- [ ] 店長メッセージをそのままLINEで送れるか
- [ ] 発言引用は正しいか
