# 録音ファイルの置き場所

ここにカウンセリング動画・音声を置いてください。

## 推奨フォルダ構成（スタッフ別）

```
input/
├── 田中/
│   ├── 2025-05-26_初回.mp4
│   └── 2025-06-02_初回.mp3
├── 佐藤/
│   └── 2025-05-28_初回.m4a
└── 山田/
    └── 2025-05-30_初回.mp4
```

## ファイル名の付け方（推奨）

```
日付_メモ.拡張子
```

例:
- `2025-05-26_初回.mp4`
- `2025-06-01_体験.m4a`

## 対応フォーマット

mp3 / m4a / wav / mp4

## 注意

- 個人名がファイル名に入っていてもOK（レポート内は必要に応じて確認）
- このフォルダのファイルは Git にコミットしないでください（録音データのため）
